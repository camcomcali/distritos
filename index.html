Cómo leer la red

No es un mapa plano

La visual no busca ubicar barrios con precisión cartográfica; busca mostrar el sistema de relaciones económicas.

  • Nodos: distritos.
  • Tamaño: masa empresarial.
  • Color: vocación dominante.

La red revela potencial

Un distrito no crece solo por estar concentrado. Crece cuando conecta oferta, demanda, anclas, aliados y soluciones.

  • Líneas suaves: relación funcional.
  • Líneas verdes: complementariedad explícita.

Del sistema al distrito

El árbol global explica el problema común. Cada distrito puede tener su propio árbol según vocación, actores y fricciones.

  • Global: modelo de ciudad.
  • Distrital: intervención específica.
